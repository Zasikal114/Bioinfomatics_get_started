## 任务2
```
library(dplyr)
species_summary <- iris %>%
  group_by(Species) %>%
  summarise(
    Mean_Sepal.Length = mean(Sepal.Length),
    SD_Sepal.Length = sd(Sepal.Length)
    )
write.csv(species_summary, "species_summary.csv", row.names = FALSE)
```
